## üîç What is DToolkits?

DToolkits helps developers handle common JSON workflows without switching between multiple websites or uploading sensitive data.

- API debugging
- JSON validation & formatting
- Type generation
- Data comparison
- Token inspection

Everything runs locally in the browser for **maximum privacy and speed**.

## üß∞ Developer Tools Included

DToolkits currently provides the following tools:

- **JSON Formatter & Validator** ‚Äì format and validate JSON instantly  
- **JSON Diff Tool** ‚Äì compare two JSON files and highlight differences  
- **JSON to TypeScript Converter** ‚Äì generate TypeScript interfaces from JSON  
- **JSON Schema Generator** ‚Äì automatically create JSON Schema  
- **JSONPath Tester** ‚Äì query and explore complex JSON structures  
- **JWT Decoder** ‚Äì decode JWT headers, payloads, and token expiry  

These tools are optimized for **real-world, large JSON data**, not just small examples.
